# Customer Churn Prediction using ANN & Streamlit

An interactive web application that predicts the likelihood of customer churn using an Artificial Neural Network (ANN) trained on banking/customer profiles. Built with **TensorFlow/Keras** and deployed using **Streamlit**.

---

## 🚀 Features

*   **Interactive Input UI:** Modify customer attributes (Credit Score, Age, Balance, Geography, etc.) via a sidebar.
*   **Real-time Machine Learning:** Live predictions powered by a deep learning Artificial Neural Network.
*   **Dynamic Visualizations:** Clean JSON output payloads and custom key metric grids.
